# REST API SPEC, URI 명명 규칙

## GET method

페이지를 렌더링하여 응답할 때 사용하는 HTTP 메소드 입니다.

- `/{프로그램명}`
    - 프로그램 인덱스 페이지
- `/{프로그램명}/{명사}`
    - 프로그램의 명사 페이지

## POST method

데이터만 응답할 때 사용하는 HTTP 메소드 입니다.

- `/프로그램명/{기능}` 또는 `/프로그램명/서브 프로그램명/{기능}`
    - 기본적으로 프로그램명을 시작으로 기능명까지 작성합니다. (패키지와 동일)
    - 기능명은 기본 명명 규칙의 메소드명 규칙에 따라 작성합니다.
